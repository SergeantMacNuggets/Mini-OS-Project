A Simple Mini Operating System that have live round robin scheduling
and memory partitioning

This program must be run only on Unix or Linux System to support the
POSIX Api's, such as pthreading. Windows or MacOS wouldn't support this
proram


How To Run:
To compile and run make sure the Makefile is in the folder, then type in the terminal:
              
            DIR/>$ make final

To run again make sure to clean first the output file, type:
          
            DIR/>$ make clean
            DIR/>% make final
